# 🎨 Pixel Collage Maker

A fun and creative **pixel-themed collage maker app** that lets users design aesthetic, retro-style collages with ease. Built with a focus on simplicity and visual appeal, this app is perfect for anyone who loves pixel art and minimal design.

# Requirements:
* Visual Studio Code (To edit your code which works easily for any coding file)
* Electron basic setup (To run your code as a desktop app which can be used anytime)
* html2canvas package (To save your collage)
* Node.js

## 🚀 Features

* 🧩 **Easy selection Interface**
  Easily add images within a flexible collage layout.

* 🎨 **Pixel-Themed UI**
  Retro-inspired design with pixel fonts and visuals for a nostalgic experience.

* 🖼️ **Image Customization**
   Layer images to create unique compositions.

* 🎯 **Grid-Based Layout**
  Align elements neatly using a structured grid system.

* 💾 **Export Option**
  Save your collage as an image for sharing or personal use.

## 🛠️ Tech Stack

* HTML
* CSS (Pixel-style design & layout)
* JavaScript (Functionality & interactivity)

## 📌 How It Works

1. Upload or select images.
2. Place them inside the collage canvas.
3. Export your final design.

## 🎯 Purpose

This app is designed for:

* Quick collage creation
* Pixel art lovers
* Beginners exploring UI-based creative tools
